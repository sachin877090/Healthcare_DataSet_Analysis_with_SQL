#  Healthcare Data Analysis using PostgreSQL

## Project Overview

#### This project focuses on analyzing a healthcare dataset using PostgreSQL. The dataset contains patient information, hospital details, medical records, and billing data.

#### The objective is to clean raw data, design a normalized database, and perform SQL-based analysis to extract meaningful insights.

---

## Tech Stack

#### PostgreSQL
#### SQL (Data Cleaning & Analysis)

---

## 📂 Dataset Description

The dataset includes the following information:

* Patient Details (Name, Age, Gender, Blood Type)
* Hospital Information (Hospital Name, Doctor)
* Medical Records (Condition, Medication, Test Results)
* Admission Details (Admission Date, Discharge Date, Room Number, Type)
* Financial Data (Billing Amount, Insurance Provider)

---

## 🏗 Database Design

The raw dataset was transformed into a normalized schema with the following tables:

### 🧑 patients

Stores unique patient information

* patient_id (Primary Key)
* name
* age
* gender
* blood_type

### 🏥 hospitals

Stores hospital and doctor details

* hospital_id (Primary Key)
* hospital_name
* doctor_name

### 📅 admissions

Stores admission and billing details

* admission_id (Primary Key)
* patient_id (Foreign Key)
* hospital_id (Foreign Key)
* admission_date
* discharge_date
* admission_type
* room_number
* billing_amount
* insurance_provider

### 💊 medical_records

Stores medical information

* record_id (Primary Key)
* patient_id (Foreign Key)
* medical_condition
* medication
* test_results

---

## ⚙️ Data Cleaning Steps

* Handled NULL values in insurance_provider
* Standardized gender values (Male/Female/Other)
* Removed negative billing records
* Capitalized patient names
* Eliminated duplicate patient entries

## 📊 Key SQL Analysis

### 🔹 Total Patients

```sql
SELECT COUNT(*) FROM patients;
```

### 🔹 Average Age

```sql
SELECT AVG(age) FROM patients;
```

### 🔹 Gender Distribution

```sql
SELECT gender, COUNT(*) 
FROM patients
GROUP BY gender;
```

### 🔹 Top Medical Conditions

```sql
SELECT medical_condition, COUNT(*) 
FROM medical_records
GROUP BY medical_condition
ORDER BY COUNT(*) DESC
LIMIT 5;
```

### 🔹 Total Revenue

```sql
SELECT SUM(billing_amount) FROM admissions;
```

### 🔹 Hospital Revenue Analysis

```sql
SELECT h.hospital_name, SUM(a.billing_amount) AS revenue
FROM admissions a
JOIN hospitals h ON a.hospital_id = h.hospital_id
GROUP BY h.hospital_name
ORDER BY revenue DESC;
```

### 🔹 Monthly Admission Trends

```sql
SELECT DATE_TRUNC('month', admission_date) AS month,
       COUNT(*) 
FROM admissions
GROUP BY month
ORDER BY month;
```
